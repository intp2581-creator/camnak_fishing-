# 심사용 안내 매뉴얼 (테스트 가이드)

> 게임물관리위원회 심사관용 · 캠피싱(CAMNAK FISHING)
> 심사관이 직접 플레이하며 확인할 수 있도록 접속·조작·주요 콘텐츠를 안내합니다.

---

## 1. 접속 방법

1. 브라우저(크롬 권장)에서 아래 주소 접속
   `https://camnak-fishing.web.app/?uid=【심사용계정이메일】`
2. 심사용 계정은 레벨·포인트·아이템을 넉넉히 지급해 두었습니다.
   - 심사 계정 : **【확인필요】 이메일 / 접속용 파라미터**
   - 지급 상태 : 고레벨 / 포인트 다량 / 민물·바다 장비 및 미끼 보유 / 아레나 입장권 보유
3. 세로형 모바일·가로형 PC 모두 16:9로 동작합니다.

> ※ 정식 서비스에서는 camnak.com 로그인 후 게임에 진입하며, 비회원은 차단됩니다.
>   심사 편의를 위해 위 파라미터로 바로 진입하도록 계정을 준비했습니다.

## 3. 주요 콘텐츠 확인 경로 (심사 포인트별)

### 3-1. 낚시 (핵심 플레이)
- 광장 → 낚시터 입장 → 미끼 장착 → 캐스팅 → 챔질 → 당기기.
- **확인 사항** : 잔혹·유혈 표현 없음. 물고기는 캐주얼 아이콘으로 표시.

### 3-2. 상점 / 결제
- 상점은 두 종류입니다.
  1) **게임 포인트 상품** : 낚싯대·릴·찌·미끼·보조장비·입장권(게임 포인트로 구매)
  2) **캐시 상품(스킨)** : camnak.com 쇼핑몰에서 현금 결제(확정 구매, 랜덤 없음)
- **확인 사항** : 유료 랜덤 뽑기(확률형 아이템) **없음**. 구매물의 현금 환전·거래 **불가**.

### 3-3. 아레나 (낚시 대회) — 사행성 확인 구간
- 광장 → 아레나 입장(무료 2회/일, 이후 입장권 사용).
- 정해진 시간 동안 최대어 크기 또는 마릿수로 순위를 겨루고, 우승자가 상금(**게임 포인트**)을 획득.
- **확인 사항**
  - 참가비·상금 모두 **게임 포인트(P)** 이며 **현금 환전 불가**.
  - 승패는 **낚시 실력(최대어/마릿수)** 으로 결정(우연·베팅 아님).
  - 아무도 못 잡으면 **무승부 → 참가비 환불**.
  - 운영 수수료 10%는 포인트 인플레 조절용(게임 내 소각).
  - 상세 근거 : 별첨 **「아레나 사행성 소명서」**.

### 3-4. 커뮤니티(채팅·친구·길드)
- 광장 하단 채팅(전체/귓속말/친구/아레나), 캐릭터 클릭으로 친구 추가, 길드 기능.
- **확인 사항** : 비속어 필터·신고·제재 정책 운영(내용정보 기술서 5항).

### 3-5. 랭킹
- 주간/월간/연간 종합 랭킹. 보상은 게임 내 요소 및 이벤트 정책에 따름.
